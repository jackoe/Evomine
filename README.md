# Evomine
Clicking is boring, evolve something to do it for you
### Compile with
```
javac MineSweeper.java -cp ../Evomine
javac EvoMine.java -cp ../Evomine
```
### Run
```
java Minesweeper
```
To play minesweeper or have it play our upper bound.

```
java EvoMine
```
To evolve a minesweeper player.
